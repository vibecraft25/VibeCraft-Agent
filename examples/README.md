# VibeCraft Agent 예제

이 디렉토리에는 VibeCraft Agent를 사용한 다양한 데이터 시각화 예제가 포함되어 있습니다.

## 📁 예제 파일

### 1. sales_data.csv
제품별 일별 판매 데이터
- 날짜, 제품, 판매량, 수익, 지역 정보 포함
- 다중 제품 비교 분석에 적합

### 2. sales_data_category.csv
카테고리별 판매 데이터
- 제품 카테고리별 집계 데이터
- 카테고리 분석 및 트렌드 파악에 유용

### 3. weather_data.json
서울시 날씨 데이터 (JSON 형식)
- 일별 기온, 습도, 풍속, 날씨 상태
- 시계열 분석 및 기상 패턴 시각화에 적합

## 🚀 사용 예제

### 기본 사용법

```bash
# CSV 파일로 막대 차트 생성
vibecraft examples/sales_data.csv "제품별 판매량 막대 차트 생성"

# JSON 파일로 시계열 차트 생성
vibecraft examples/weather_data.json "날짜별 기온 변화 라인 차트 생성"

# 스트리밍 모드로 실행 과정 확인
vibecraft -s examples/sales_data.csv "지역별 수익 파이 차트 생성"
```

### 고급 사용법

#### 1. 복합 차트 생성
```bash
vibecraft examples/sales_data.csv "제품별 판매량과 수익을 함께 보여주는 복합 차트 생성. 막대 그래프로 판매량을, 라인 그래프로 수익을 표시"
```

#### 2. 대시보드 생성
```bash
vibecraft examples/sales_data.csv "전체 판매 현황 대시보드 생성. 다음 차트들을 포함: 1) 일별 총 수익 추이, 2) 제품별 판매량 비교, 3) 지역별 수익 분포, 4) 제품-지역별 히트맵"
```

#### 3. 날씨 데이터 분석
```bash
vibecraft examples/weather_data.json "날씨 종합 분석 차트 생성. 기온 변화 라인 차트와 습도 막대 그래프를 결합하고, 날씨 상태를 색상으로 표시"
```

## 📊 예상 결과

각 명령어는 `vibecraft-dashboard.html` 파일을 생성합니다. 이 파일을 웹 브라우저에서 열면:

- 인터랙티브한 차트 (확대/축소, 데이터 포인트 hover)
- 반응형 디자인 (모바일/데스크톱 대응)
- 다양한 차트 타입 (막대, 라인, 파이, 히트맵 등)
- 전문적인 스타일링과 애니메이션

## 💡 팁

1. **자연어 프롬프트**: 구체적이고 명확한 요구사항을 전달하면 더 좋은 결과를 얻을 수 있습니다.

2. **스트리밍 모드**: `-s` 옵션을 사용하면 AI가 데이터를 분석하고 차트를 생성하는 과정을 실시간으로 볼 수 있습니다.

3. **디버그 모드**: 문제가 발생하면 `-d` 옵션으로 상세한 디버그 정보를 확인할 수 있습니다.

4. **출력 파일 지정**: `-o` 옵션으로 다른 이름의 HTML 파일을 생성할 수 있습니다:
   ```bash
   vibecraft -o sales_report.html examples/sales_data.csv "판매 보고서 생성"
   ```

## 🎨 시각화 아이디어

### 판매 데이터
- 시계열 분석: 일별 판매 추이
- 제품 비교: 제품별 성과 분석
- 지역 분석: 지역별 판매 패턴
- 상관관계: 판매량과 수익의 관계

### 날씨 데이터
- 기온 추이: 최고/최저/평균 기온 변화
- 기상 패턴: 날씨 조건별 분포
- 상관분석: 기온-습도-풍속 관계
- 종합 대시보드: 모든 기상 요소 한눈에

## 📝 커스텀 데이터 사용

자신의 데이터로 시각화를 생성하려면:

1. CSV 또는 JSON 형식으로 데이터 준비
2. 명확한 컬럼명/필드명 사용
3. 일관된 데이터 형식 유지
4. 적절한 시각화 요구사항 작성

예시:
```bash
vibecraft my_data.csv "매출 데이터를 분석하여 트렌드와 이상치를 찾고, 예측 모델을 포함한 대시보드 생성"
```
